## 阶段 1：单机核心玩法（沙盒 + 科技生产线）

**目标**：单人模式下完成资源采集 → 建造 → 自动化生产的完整循环。

### 1.1 资源系统
- [ ] 定义基础资源类型（矿石、植物、旧世界残骸）
- [ ] 实现资源拾取、存储（背包组件）
- [ ] 放置资源节点（矿脉、废料堆）

### 1.2 建造系统
- [ ] 简易放置/拆除建筑（地基、传送带、熔炉、组装机）
- [ ] 建筑数据序列化（本地存档 JSON）
- [ ] 建筑与资源交互（熔炉烧矿）

### 1.3 自动化生产线核心
- [ ] 传送带物流：物品在传送带上移动，相邻建筑自动输入/输出
- [ ] 配方系统：定义配方（输入 → 输出 + 耗时）
- [ ] 生产设备逻辑：熔炉、组装机按配方处理物品
- [ ] 存储箱/缓冲箱

### 1.4 科技树（基础版）
- [ ] 科技解锁条件（消耗特定资源或研究点数）
- [ ] ImGui 原型 UI 展示科技树

### 1.5 单机存档与加载
- [ ] 完整保存世界状态（所有实体组件数据）
- [ ] 支持手动保存/自动保存

**里程碑**：单人完成「采集矿石 → 烧成金属板 → 加工成齿轮 → 建造更高级机器」的完整链条。

---

## 阶段 2：多人联机完善（协作与同步）

**目标**：2~4 人稳定合作建造，共享世界，状态一致。

### 2.1 房间/世界管理
- [ ] 创建/加入房间（服务器权威）
- [ ] 世界数据托管：服务器存储 ECS 快照

### 2.2 状态同步优化
- [ ] 增量同步（只发送变化的组件）
- [ ] 兴趣管理（AOI，只同步视野内实体）
- [ ] 同步频率自适应

### 2.3 多人建造同步
- [ ] 建造/拆除请求经服务器验证后广播
- [ ] 冲突处理（多人同时操作同一位置）

### 2.4 断线重连
- [ ] 客户端断线后保留玩家实体一段时间
- [ ] 重连时下载全量世界状态

### 2.5 战斗基础（轻量）
- [ ] 简单攻击动作（左键点击目标）
- [ ] 血量与伤害同步（服务器权威）
- [ ] 敌人 AI（巡逻、攻击设施）

**里程碑**：多人可稳定在同一世界中建造、采集、战斗，无严重卡顿或不同步。

---

## 阶段 3：科技生产线深度（GTNH 复杂度）

**目标**：多步加工、流体、电力、跨区域物流，实现高复杂度自动化。

### 3.1 多层级资源
- [ ] 中级资源：合金、电子元件、管道
- [ ] 高级资源：量子元件、时空稳定器、记忆碎片

### 3.2 流体系统
- [ ] 液体（水、油、冷却液）储存与管道运输
- [ ] 流体与生产设备交互（化工厂）

### 3.3 电力系统
- [ ] 发电机、电线杆、电池
- [ ] 设备耗电、负荷管理

### 3.4 物流扩展
- [ ] 无人机、火车、空间传送门
- [ ] 跨区域物流站

### 3.5 科技树完整
- [ ] 分支：能源、物流、战斗、旧文明覆写科技
- [ ] 研究点数自动生产（研究实验室）

### 3.6 性能优化
- [ ] 多线程生产线计算
- [ ] LOD 渲染与实体剔除

**里程碑**：搭建类似 GTNH 早期自动化产线（「铁 → 螺丝 → 传送带 → 电路板 → 基础计算机」）。

---

## 阶段 4：二游与社区内容集成

**目标**：角色、抽卡、配饰、开源贡献奖励系统的完整经济循环。

### 4.1 角色系统
- [ ] 可获取助手角色（抽卡/任务）
- [ ] 角色外观、简单 AI（跟随、采集、战斗）

### 4.2 抽卡系统
- [ ] 抽卡券由高级生产线产出（科技驱动抽卡）
- [ ] 服务器端抽卡逻辑（防作弊）
- [ ] 卡池配置（角色皮肤、建筑皮肤、配饰）

### 4.3 配饰与覆写层数
- [ ] 配饰数据从废墟探索或抽卡获得
- [ ] 覆写层数解锁（探索 + 贡献 + 生产）
- [ ] UI 展示配饰故事文本

### 4.4 开源贡献兑换
- [ ] GitHub OAuth 绑定
- [ ] Webhook 监听 PR/Issue，计算贡献值
- [ ] 贡献值兑换抽卡券或专属配饰

### 4.5 碎片剧情系统
- [ ] 可扫描的记忆残骸（废墟中分布）
- [ ] 播放文本/音频片段，解锁配饰层数

**里程碑**：经济闭环——贡献代码 → 获得抽卡券 → 抽到新皮肤 → 在多人世界中展示。

### 5.4 社区工具

### 5.5 发布

- [ ] Bug 跟踪与贡献指南
- [ ] v1.0 正式发布

## 时间预估

| 阶段 | 工作量（人月） | 说明 |
|------|:----------:|------|
| 阶段 1 | 2-3 | 单人核心玩法 |
| 阶段 2 | 2 | 多人联机基础 |
| 阶段 3 | 3-4 | GTNH 复杂度 |
| 阶段 4 | 2 | 二游与社区系统 |

总计约 9-12 人月达到可公开测试的完整版。贡献者越多，周期越短。

## 当前建议

已完成渲染和基础联机，**下一步集中精力完成阶段 1 的核心玩法**（资源 → 建造 → 自动化）。这是游戏最独特的部分，也是社区最感兴趣的内容。单人玩法可玩后，再回头加强多人联机。
