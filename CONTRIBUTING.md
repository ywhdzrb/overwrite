# 贡献指南

感谢你考虑为 OverWrite 做出贡献！

## 如何贡献

- **代码** - 提交 Pull Request
- **创意** - 在 Issues 中提交剧情/活动提议
- **反馈** - 报告 Bug 或提出建议
- **Wiki** - 编写文档和攻略

## 贡献者许可协议

通过向本项目提交 Pull Request 或 Issue（包含代码、模型、贴图、文档等任何内容），您同意：

1. 您所贡献的内容必须是您原创或已获得合法授权。
2. 您将贡献的内容以**与项目对应部分相同的许可证**授予本项目：
   - 代码贡献采用 GPLv3
   - 美术资产贡献采用 CC BY-NC-SA 4.0
3. 您同意在贡献内容中署名。

如果您不同意上述条款，请不要提交贡献。

## 贡献者奖励

- 抽卡资源奖励
- 专属配饰/角色（含贡献者 ID 刻印）
- 更新日志特别鸣谢
